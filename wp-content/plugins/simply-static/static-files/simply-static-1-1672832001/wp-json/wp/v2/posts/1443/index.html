PL\/SQL is a block structured language that enables developers to combine the power of SQL with procedural statements.All the statements of a block are passed to oracle engine all at once which increases processing speed and decreases the traffic.<\/p>\n\n\n\n

Disadvantages of SQL:<\/strong><\/p>\n\n\n\n

  • SQL doesn\u2019t provide the programmers with a technique of condition checking, looping and branching.<\/li>
  • SQL statements are passed to Oracle engine one at a time which increases traffic and decreases speed.<\/li>
  • SQL has no facility of error checking during manipulation of data.<\/li><\/ul>\n\n\n\n

    Features of PL\/SQL:<\/strong><\/p>\n\n\n\n

    1. PL\/SQL is basically a procedural language, which provides the functionality of decision making, iteration and many more features of procedural programming languages.<\/li>
    2. PL\/SQL can execute a number of queries in one block using single command.<\/li>
    3. One can create a PL\/SQL unit such as procedures, functions, packages, triggers, and types, which are stored in the database for reuse by applications.<\/li>
    4. PL\/SQL provides a feature to handle the exception which occurs in PL\/SQL block known as exception handling block.<\/li>
    5. Applications written in PL\/SQL are portable to computer hardware or operating system where Oracle is operational.<\/li>
    6. PL\/SQL Offers extensive error checking.<\/li><\/ol>\n\n\n\n
